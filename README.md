# Web3 能否改变中国漫展行业？  
## ——一位漫展从业者与 Web3 学习者的结构性思考

过去两年，中国漫展行业进入了一个极具张力的阶段。

一方面，漫展行业呈现出几乎肉眼可见的爆发式增长：  
泛二次元用户规模已超过 **5 亿**；  
到 **2025 年**，全国范围内漫展及相关活动数量突破 **1000 场**；  
BW、CP 等头部展会依旧一票难求，开票即秒空。

但另一方面，真正身处行业内部的人，往往会给出一个更冷静的判断：

> 漫展行业看起来前所未有地繁荣，但结构性的疲态已经非常明显。

作为一名长期参与漫展策划与执行的从业者，同时也是一名持续关注 Web3 的学习者，我越来越清楚地意识到——  
**中国漫展行业的问题，从来不在“热不热”，而在“结构是否还能继续承压”。**

也正是在这样的背景下，“Web3 + 漫展”开始被反复提起。  
有人将其视为新风口，有人认为是概念包装，也有人担心这会成为下一轮资本围猎。

但真正值得讨论的问题并不是：  
**要不要 Web3？**  
而是：  
**在 Web3 技术可落地、合规可执行的前提下，它是否能够修复 Web2 体系中一些长期被默认、却始终无解的行业结构问题？**

---

## 一、漫展行业真正的问题：不是规模，而是结构失衡

如果只看数据，中国漫展行业几乎无可挑剔。

以头部同人展 **CP31** 为例，单场参展人次超过 **16 万**，门票均价约 **120 元**，仅门票收入就接近 **2000 万元**。  
放眼全国，中大型漫展的收入规模早已进入千万级。

但在实际执行层面，行业正面临至少三重越来越尖锐的矛盾。

### 1️⃣ 漫展数量增长，远快于有效需求增长

近两年，漫展活动呈现出明显的“过密化”趋势。

同一城市、同一时间段内，多场漫展同时举办；  
原本一年一次的区域性展会，被拆分为一年多次。

结果是：

- 头部展会依旧爆满  
- 大量中小型漫展客流被严重分流  
- 盈亏线不断被拉高  

这并不是简单的“竞争加剧”，而是**供给扩张速度已明显快于核心二次元用户的增长速度**。

---

### 2️⃣ 成本持续抬升，但主办方逐渐失去定价权

在当前 Web2 体系下，漫展商业模型高度依赖两个核心变量：

- **平台流量**  
- **嘉宾吸引力**  

但这两端，主办方都不掌握主动权。

以票务与宣发平台为例，国内主流平台对漫展类活动的综合抽成比例，通常在 **10%–20%** 之间浮动。  
这意味着，一场门票收入为 **1000 万元** 的漫展，仅平台服务费一项，就可能被抽走 **100–200 万元**。

更关键的是，这笔成本并不会换来“用户资产”：

- 购票数据归平台  
- 用户关系归平台  
- 复购路径依旧需要重新投流  

主办方实际上是在进行一场又一场的**流量租赁**。

与此同时，嘉宾体系也在被高度金融化。

五年前，头部 Coser 的单场出场费多集中在数万元级别；  
而随着 MCN 深度介入、短视频平台以粉丝量和互动量作为核心定价指标，部分顶流 Coser 的商业报价已进入 **十万甚至百万级区间**。

有业内人士透露，某知名 Coser 的一次商业快闪活动报价接近 **100 万元**，且不包含服装、差旅与现场执行成本。

在这种结构下，嘉宾更像是一种高价流量商品，而主办方承担了几乎全部的商业风险。

---

### 3️⃣ 行业参与度极高，但关系几乎无法沉淀

这是当前漫展行业最容易被忽视、却最根本的问题。

在 Web2 体系下：

- 观众属于平台  
- 数据属于平台  
- 社群关系散落在群聊与账号中  

即便连续多年举办同一 IP 漫展，主办方依然需要从零开始争夺注意力。

**漫展办得越多，沉淀反而越少。**

---

## 二、先划清边界：Web3 不能解决什么？

我们在讨论 Web3 之前，必须先清除不切实际的幻想。

Web3 并不能解决：

- 二次元人口红利见顶  
- 内容同质化  
- 嘉宾短期溢价  
- 区域市场本身的承载能力不足  

如果指望 Web3 “拯救所有漫展”，那一定会失败。

但 Web3 **有可能介入并改善**三类 Web2 长期无解的问题：

1️⃣ **身份与参与无法沉淀**  
2️⃣ **规则执行高度依赖平台**  
3️⃣ **小型社群与 Only 展难以长期存活**

---

## 三、Web3 介入后，Web2 旧局面可能发生的变化

### （一）票务系统：从一次性交易，变成参与关系的记录层

目前，漫展的票务系统本质上依赖平台，且票务购买与用户身份没有长期关联。每次活动的票务销售，都是一笔“独立交易”，不会留下长期的用户关系。然而，Web3 提供了一个**可验证、可追溯的身份层**，使得票务不仅是一次性交易，而是可以与观众的长期参与和忠诚度相结合。

#### **如何实现：**
- **NFT 票务系统**：Web3 可以通过 **NFT**（非同质化代币）来发行门票。每一张门票都可以是一个独特的数字资产，绑定参与者的身份信息，这个门票在活动结束后可以留存，并作为其“参与历史”的一部分。例如，参与了某一 IP 漫展的用户，可以通过 NFT 来记录其参与历史，并在未来的展会中获得优先购买权或独特权益。
  
- **粉丝积分与奖励**：除了基本的门票，Web3 可以进一步通过智能合约设计粉丝积分系统，根据用户在漫展中的活动（比如购买周边、参与互动等）给予相应的积分或奖励。这些奖励可以作为下一次活动的折扣、独家内容或早期入场权限，进一步增强用户粘性。

这种方式不仅减少了黄牛的作用，也增强了主办方对用户关系的长期掌控。

---

### （二）黄牛问题：从技术对抗，转向规则设计

目前的票务系统无法有效遏制黄牛行为。即便平台采取技术手段限制黄牛的活动，黄牛依然可以通过各种方法规避这些限制。而 Web3 的核心优势之一，就是去中心化和透明性，能够通过智能合约和链上规则设计，显著减少黄牛的行为。

#### **如何实现：**
- **价格上限与二级市场控制**：Web3 可以通过智能合约设置票务的价格上限，限制转售价格，避免黄牛大规模倒卖。主办方可以规定，门票转售时不得超过原价的某个比例，例如不超过 10% 的溢价。
  
- **透明的转售市场**：通过链上交易，票务的转售信息将完全透明，任何一次转售行为都可以在链上追溯，确保所有交易都是公开的，杜绝暗中操控。
  
- **官方回购机制**：主办方可以设置一个回购机制，允许在票务转售价格上涨过快的情况下，主办方有权在链上回购票务，并退还原买家的资金。这一机制可以在转售价格异常时，及时进行干预，保持市场秩序。

---

### （三）创作者与嘉宾：从一次性交易，到长期价值参与

在传统的漫展中，创作者与嘉宾的合作多为一次性交易，嘉宾的出场费往往是一次性的，高昂的价格并没有给创作者带来长期收益。通过 Web3，创作者和嘉宾的收入可以与其长期价值绑定，并且通过智能合约确保创作者在作品二次销售中得到持续分成。

#### **如何实现：**
- **数字藏品与收入分成**：Web3 技术使得每一件数字周边（如数字画作、虚拟商品）都可以被视作唯一资产，并与创作者绑定。创作者不仅能从首次销售中获益，还可以从二级市场的转售中获得持续收益。例如，某位插画师的数字作品通过 Web3 方式被绑定到 NFT，用户购买后如果该作品被转售，创作者可以通过智能合约获得 **5%–10%** 的转售抽成。

- **创作者社群治理**：通过 **DAO**（去中心化自治组织），创作者可以参与漫展的管理与决策过程。DAO 模式允许所有参与的创作者、粉丝及相关方一起讨论和决策，推动创作者社区的健康发展。

这种模式不仅让创作者从单次交易中获益，更将他们的持续参与与价值创作带入到 Web3 的长期生态中。

---

### （四）Only 展：从“一次活动”，进化为“长期组织”

Only 展往往是最有文化黏性的形态，却也是最脆弱的。由于缺乏长期的社群积累和用户关系，很多 Only 展无法持续发展。而 Web3 可以通过去中心化的方式让这些小型展会保持活跃性，并让参与者感到更加连接和忠诚。

#### **如何实现：**
- **去中心化社群管理**：Web3 提供了去中心化的身份认证和社群管理模式。通过去中心化平台，Only 展可以将其参与者的身份信息、历史参与记录以及贡献情况保存在链上，从而实现更长期、更健康的社群运营。

- **长期参与权益**：参与者不仅可以通过传统的门票参与到展会中，还可以通过贡献记录获得社群治理权、未来展会的优先权或定制化权益。

Web3 为 Only 展提供了建立长期可持续发展的基础，使得这些小型展会能够从“短期一锤子买卖”转向长期的社群文化创造。

---

## 四、必须正视的风险：治理疲劳与技术门槛

虽然 Web3 为漫展行业带来了巨大的潜力和优势，但也存在着潜在的风险。

### 1. **治理疲劳**

DAO 机制需要足够的参与者和活跃度，但随着社群的扩展，治理结构可能面临决策效率低下、治理疲劳等问题。过多的参与者和决策可能导致协作困难。

### 2. **技术门槛**

Web3 技术的实现需要强大的技术支持，而漫展行业的从业者可能缺乏足够的技术基础。这使得 Web3 技术在初期的推广和实施中，可能面临较高的学习曲线和技术挑战。

因此，在中国语境下，Web3 更适合作为：

> **数字确权 + 身份系统 + 规则执行工具**  
而不是金融产品。

只有在不增加普通参与者负担的前提下，Web3 才可能成为基础设施，而非新的噱头。


---

## 结语：Web3 不是答案，但可能让行业不再拧巴

作为漫展从业者，我并不认为 Web3 能让所有漫展更赚钱。

但它至少让行业第一次有机会回答一些长期被忽略的问题：

- **谁是真正的核心用户？**  
- **谁在持续创造价值？**  
- **谁值得被优先尊重与回馈？**  

而作为 Web3 学习者，我也越来越确信：  
如果 Web3 连这样一个高度社群化、情绪化、创作者密集的行业都无法改善，那它在更复杂的现实世界中，也很难真正站住脚。

Web3 或许无法改变漫展的热闹，  
但它可能让漫展重新拥有**秩序、记忆与延续性**。

这，或许才是它最值得尝试的地方。
